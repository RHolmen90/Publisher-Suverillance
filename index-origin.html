Features

This is a npm package to display statistics about your web pages, including Google PageSpeed Insights score, resources count, recommendations how to fix performace issues, html errors and many more in one custom web page.

Test multiple pages on Google PageSpeed Insights at once

With PerfTool you can run multiple tests for Google PageSpeed Insights report. Then, compare results from previous tests, set score threshold and get detailed report about your tests.

View Sample Report

Get your page statistics

Check your page for html errors, get external resources count and sizes, check loading times and many more.

Over 200 options

Installation

PrefTool package is combination of three plugins: Google PageSpeed Insights - to test page performance, dev-perf - to get statistics about your page, and w3cjs - to validate html.

npm install devbridge-perf-tool --save-dev require('gulp').task('perf-tool', function () { var options = { siteURL:'http://www.google.com', sitePages: ['/', '/voice'] }; return require('devbridge-perf-tool').performance(options); });

More information about installation process and how to use it on our github page.
