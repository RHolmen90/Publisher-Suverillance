Features

This is a npm package to display statistics about your web pages, including Google PageSpeed Insights score, resources count, recommendations how to fix performace issues, html errors and many more in one custom web page.

Test and compare scores from Google PageSpeed Insights on one page

With PerfTool you can test multiple pages on Google PageSpeed Insights at once. When you can compare your results with previous tests. In addition you can set your threshold for scores.

View Sample Report

Check your page’s HTML healt

From html errors to external assets count, check your page’s health. You can customize your results from over 200 options.

Over 200 options

Installation

PrefTool package is combination of three plugins: Google PageSpeed Insights - to test page performance, dev-perf - to analyse html, and w3cjs - html validator.

npm install devbridge-perf-tool --save-dev
require('gulp').task('perf-tool', function () { var options = { siteURL:'http://www.google.com', sitePages: ['/', '/voice'] }; return require('devbridge-perf-tool').performance(options); });

More information about installation process and how to use it on our github page.
